GoFood Automation Framework is a scalable Selenium-based test automation project developed to validate the end-to-end functionality of the GoFood web application. The framework follows industry-standard design principles, enabling maintainable, reusable, and efficient automated testing.

Tech Stack
Java
Selenium WebDriver
TestNG
Maven
Page Object Model (POM)
Page Factory
Extent Reports
Log4j / SLF4J Logging
Apache POI (Excel Data-Driven Testing)
Git & GitHub
Jenkins (CI/CD Ready)
Framework Features
End-to-end automation of GoFood web application
Cross-browser execution (Chrome & Safari)
Parallel execution using TestNG
Data-driven testing with Excel
Explicit waits and reusable utility classes
Dynamic element handling
Configuration through Properties files
Rich HTML reports using Extent Reports
Screenshot capture on test failures
Retry mechanism for flaky tests
Thread-safe WebDriver management using ThreadLocal
Modular Page Object Model architecture
Easy integration with Jenkins for Continuous Integration
